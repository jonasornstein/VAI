## 1. Overview and schedule

**V85®** is ATG's flagship **Saturday Pick8** pool for trotting (*trav*). Pick the **winner** in **8 predetermined races** (*avdelningar*). Win on **8, 7, 6, or 5** correct legs.

| Attribute | Value |
|-----------|-------|
| Legs | 8 |
| Ordinary schedule | Saturdays (few exceptions) |
| Extra rounds | Some weekdays and Sundays |
| Row price (*radpris*) | **0.50 SEK** per combination |
| Return to players | 65% |
| Win tax (*vinstavdrag*) | 35% |
| Premiere | 2025-10-25, Jägersro |

**References:** [atg.se/v85](https://www.atg.se/v85) · [ATG kundservice — V85](https://atg-extern.kb.kundo.se/guide/information-angaende-v85)

## 2. How to play

- Mark **one or more horses** per leg (*gardering*); one horse = *spik*.
- Win if your selections include the **official winner** in each leg counted toward your prize level.

### 2.1 System cost

```
rows (combinations) = ∏ (horses marked in leg i)  for i = 1..8
cost (SEK)          = rows × 0.50
```

**Example:** 1 × 3 × 1 × 2 × 1 × 4 × 1 × 2 = **48 rows** → **24.00 SEK**.

| Swedish | English |
|---------|---------|
| Rad | Row (one combination) |
| System | Full set of rows |
| Kupong | Coupon |
| Avdelning | Leg |

## 3. Payout pools (from 2026-07-02)

Effective Sprintermästaren **2026-07-02** (Halmstad).

| Correct | Pool share | Jackpot |
|---------|------------|---------|
| **8** | **30%** | No winners or payout **&lt; 5 SEK** → jackpot (next ordinary round) |
| **7** | **20%** | — |
| **6** | **15%** | Payout **&lt; 7 SEK** (after any 5-correct top-up) → jackpot / dubbeljackpot |
| **5** | **35%** | **No jackpot**; payout **&lt; 5 SEK** → funds move to **6-correct pool** |

**Historical (2025-10-25 – 2026-07-01):** 35 / 15 / 15 / 35 with **5 SEK** jackpot threshold on all pools.

**Carry-over:** Jackpot funds roll to the **highest prize pool** on the next ordinary round (S-008). **MultiJackpot:** 20% of ordinary-round jackpots + 100% of extra-round jackpots fonderas to 8-correct when ATG decides (S-014).

## 4. Reduced stake (*sänkt insats*)

Mathematical systems only. Payout scales with stake (30% / 50% / 70% of ordinary dividend).

| Limit | Value |
|-------|-------|
| Max coupons per customer / round | **3** |
| Max rows per coupon | **4 000** |
| Max systems per account / round | **5 000** |

Not eligible: butiksandelar, Tillsammans, reducerat spel, Harry Boy.

## 5. Edge cases

### Scratched horses (*struken*)

- Up to **two reserves** per leg (optional; else **established reserve order**).
- Customer reserves apply in **priority order** on withdrawal (S-009a §4–§6).

### Dead heat (*dött lopp*)

- Each horse tied for 1st is a **winner** (S-009a §10).
- Marking multiple dead-heat winners **multiplies** winning combinations.

### Disqualification (*utkörning*)

- Betting result = Arranger's **definitive result** (S-009a §20).

### Cancelled races

- Moved within **24 hours** → not cancelled.
- Cancelled game → **void**, stakes refunded (S-006).

## 7. ATG entry (operators)

1. Open [atg.se/v85](https://www.atg.se/v85) on race day.
2. Build system leg by leg; set reserves if needed.
3. Verify **row count** and **SEK cost** match proposal.
4. Check **scratches** before start.
5. Submit manually — this project does **not** automate placement.

---

## 8. Proposal requirements (this project)

- All **8 legs** with **start numbers**
- **Row count** and **total SEK cost**
- **Track, date**, and **mode**
- Scratch/reserve reminder for race day
